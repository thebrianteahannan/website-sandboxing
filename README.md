# Gay-o-Meter

Paste a chat snippet. Get a theatrical reading on how gay the vibes are.

## Run it

Open `index.html` in a browser, or from this folder:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## How it works

Client-side only. Text never leaves your browser. Scoring looks for slang, punctuation drama, emoji cocktails, and a few culture references — then maps that onto a 0–100 gauge.

Purely for entertainment.
